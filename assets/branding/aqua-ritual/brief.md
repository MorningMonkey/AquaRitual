# Branding Brief: AquaRitual (aqua-ritual)

## 1. コンセプト（1文）
水槽の育成体験で、日々の習慣化（ストリーク）を静かに継続できるアプリ。
Tagline: Build habits. Grow calm.

## 2. ターゲット
- 主対象: 一般ユーザー
- 利用シーン: 集中して作業に取り組むとき（試験勉強や没頭作業）、集中そのものを習慣化したいとき
- 期待価値: 習慣の継続、癒し、集中の維持

## 3. 想定プラットフォーム
- 対象: iOS / Web（PWA含む）
- 最優先: iOS
- 画面前提: モバイル優先 / レスポンシブ対応

## 4. トーン&マナー（言葉遣い）
- 基調: 優しい・肯定的
- 出力言語: 日本語を正（英語は固有名詞・UI語彙のみ）
- 禁止: 否定的・悲観的な表現、長文

## 5. ビジュアル指針（UI/画像）
- キーワード: 集中(Concentration), 習慣化(Habit), 癒し(Healing)
- 色/質感: 明るめ・淡い色・透明感 / ガラス、柔らかい光、控えめなグラデーション
- 形/情報密度: 丸み（泡/水滴）、緩やかな波形 / 高可読・余白重視・過度に装飾しない

## 6. モチーフ（抽象）
- OK: 水の循環、泡、水面の反射、静かな光度 / メタファー「毎日の習慣が水槽を育てる」
- NG: 和風・高級・特定作品連想、特定キャラクター

## 7. README/成果物の表現ルール
- 結果報告は日本語で「完了内容 / 生成・更新ファイル / 次のアクション」を含める
- 「結果は walkthrough.md を参照」の一文だけで終えない（本文に要点を要約）
